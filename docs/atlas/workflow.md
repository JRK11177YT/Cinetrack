# Workflow de trabajo con Atlas

## 1. Objetivo

Este documento define cómo se utilizará Atlas como sistema de apoyo técnico durante el desarrollo de CineTrack.

Atlas no forma parte de la aplicación final, sino del método de trabajo del proyecto.

---

## 2. Principio de uso

Atlas se utilizará como guía especializada para tomar decisiones técnicas con orden, coherencia y enfoque profesional.

Cada fase del proyecto se desarrollará con el apoyo del modo de Atlas más adecuado.

---

## 3. Modos de uso

### Architect

Se utilizará para:

* estructura general del proyecto
* arquitectura MVC
* organización de paquetes y capas
* decisiones de diseño

### DevOps

Se utilizará para:

* Docker
* docker-compose
* variables de entorno
* comunicación entre servicios
* despliegue local

### DBA

Se utilizará para:

* diseño de base de datos
* modelo ER
* modelo relacional
* tablas, relaciones y scripts SQL

### Git Lead

Se utilizará para:

* estrategia de ramas
* convenciones de commits
* control de versiones
* definición de hitos y releases

### Tribunal

Se utilizará para:

* simulación de defensa oral
* revisión crítica
* detección de debilidades
* mejora de justificaciones técnicas

---

## 4. Flujo de trabajo previsto

### Fase 1 — Organización e infraestructura

Modos principales:

* Git Lead
* DevOps

### Fase 2 — Base de datos

Modos principales:

* DBA
* Architect

### Fase 3 — Base del backend

Modos principales:

* Architect
* DevOps

### Fase 4 — Desarrollo funcional

Modos principales:

* Architect
* DBA
* Git Lead

### Fase 5 — Documentación y defensa

Modos principales:

* Tribunal
* Architect
* Git Lead

---

## 5. Beneficios esperados

El uso de Atlas permitirá:

* mantener orden técnico
* evitar improvisación
* justificar decisiones con mayor claridad
* mejorar la calidad global del proyecto
* preparar una defensa más sólida
* desarrollar CineTrack con enfoque profesional

---

## 6. Filosofía

Atlas se utilizará para construir un proyecto excelente, no solo correcto.

La prioridad será siempre la calidad, la coherencia y la profesionalidad del resultado final.
