# CineTrack

Proyecto transversal final DAW/DAM.

## Descripción
CineTrack es una aplicación web monolítica inspirada en una plataforma de streaming de películas, desarrollada como proyecto transversal final.

## Tecnologías previstas
- Java
- Spring Boot
- Thymeleaf
- Bootstrap 5
- MySQL
- Docker
- GitHub

## Estado del proyecto
Fase inicial de planificación, infraestructura y organización.
